Настройка и проверка PyTorch для win10

Качаем и ставим анаконду 3.6:
https://www.anaconda.com/download/#windows

Переменной anaconda3_path не существует, но представим что это так
Запускаем командную строку (Anaconda Prompt) с нужными переменными
> %windir%\System32\cmd.exe "/K" %anaconda3_path%\Scripts\activate.bat %anaconda3_path%
P.S. Anaconda Prompt можно найти и в пуске, но не понт

Ставим в этой консоли Pytorch (инструкция есть прям на главной http://pytorch.org/, чтобы для своего железа скачать то, что надо)
> conda install pytorch -c pytorch
> pip3 install torchvision

Качаем https://github.com/mrfeod/examples/blob/mnist/mnist/main.py
Код тренировщика пофикшен со сраного линукса для православной винды

Запускаем обучение
> python main.py

Ждем несколько минут пока пройдет 10 эпох и если все удалось - PyTorch настроен, можно гонять лысого нейронки
